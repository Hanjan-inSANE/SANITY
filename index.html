Threat scenarios (RAG-grounded)

Click Derive scenarios: the engine queries the Attack-RAG threat DB once per component (using each component's configuration) to gather TTP / CVE / CWE evidence, then the LLM infers plausible scenarios — each an objective → one or more entry components → one endpoint — grounded in that evidence.
